# StayBusy — UI kit

A full, interactive recreation of the StayBusy iOS app inside an iPhone frame,
composing the design-system components from `components/core/`.

## Files
- `index.html` — entry point. Loads React, the compiled `_ds_bundle.js`,
  Phosphor icons, the iOS frame, and the three screen modules, then mounts the
  app. Open this.
- `sb-data.js` — sample schedule (a touring musician's two days), built
  relative to "today" so the Now/Next bar and now-line stay live.
- `sb-screens.jsx` — **Today** (hour-ruled proportional timeline + Now/Next bar
  + floating add) and **Trip** (multi-day overview with density bars).
- `sb-detail.jsx` — **Map** (numbered pins + bottom card), **Block detail**
  (confirmation code, notes, links, Leave-by), **Editor** sheet, the bottom
  **tab bar**, and the app shell.

## What's interactive
- Switch tabs (Today / Map / Trip).
- Step days with the picker; Jump to Today.
- Tap a timeline block → detail view; tap the confirmation code → Copied.
- Tap an OPEN slot or the **+** → editor sheet (pick a category, Save/Cancel).
- Map: tap a numbered pin → bottom card → Details.
- Trip: tap a day → jumps to that day's timeline.

## Notes
- Requires the compiled `_ds_bundle.js` at the project root (generated by the
  design-system compiler). Components are read from
  `window.StayBusyDesignSystem_0220d6`.
- The map is a stylized flat surface (the real app uses Apple MapKit); the
  Leave-by ETA is illustrative.
